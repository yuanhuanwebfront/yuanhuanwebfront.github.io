# grid布局

grid是一个CSS的简写属性。可以实现网格布局。之后使用 grid-template-rows（行属性） 和 grid-template-columns（列属性） 属性来定义。

# grid-template-rows

grid-template-rows 该属性是基于网格行的维度，去定义网格线的名称和网格轨道的尺寸大小。
