# 包管理

# verdaccio 官网 (opens new window)

对于较大的团队，对于私有仓库的需求是很必要的。私有仓库可以在公司内部共享npm包。 verdaccio的安装和使用都比较简洁，以下记录使用过程。

# 使用npm安装

  npm install --location=global verdaccio

安装完成后执行如下命令查看是否安装成功

  verdaccio

# 开始使用

启动服务后，需要重新设置registry，从而可以下载我们自己的仓库文件

  npm set registry http://localhost:4873/

之后为仓库添加用户

  npm adduser --registry http://localhost:4873/

按照提示完成账号、密码、邮箱的添加。

之后我们新建一个项目 将项目推送，推送前请先登录刚才添加的账号

  npm login

发布刚才添加的项目

  npm publish --registry http://localhost:4873/

访问 http://localhost:4873/ 可以看到 刚才的包已经发布成功了。
